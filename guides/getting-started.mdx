## Understanding environments
Ramp offers two environments:
<Tabs>
  <Tab title="Sandbox">
| Base URL                    | Description                                                                 |
|-----------------------------|-----------------------------------------------------------------------------|
| `https://demo-api.ramp.com` | For testing and development. Contains demo data. Apps created here only work in this environment. |
  </Tab>
  <Tab title="Production">
| Base URL                | Description                                                                 |
|-------------------------|-----------------------------------------------------------------------------|
| `https://api.ramp.com`  | For live use. Interacts with your real Ramp account. Use only after testing in Sandbox. |
  </Tab>
</Tabs>
